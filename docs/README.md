
# docs

本資料夾提供與本專案相關之補充文件、快速指南與圖示。

## 結構

```text
docs/
├─ quick_start.md
├─ reproducibility_checklist.md
├─ diagrams/
│  ├─ tiaa_sam_stride_mermaid.md
│  ├─ stride_pipeline_mermaid.md
│  ├─ pdm_reasoning_chain_mermaid.md
│  └─ repo_structure_mermaid.md
└─ README.md
```

- `quick_start.md`：提供從零開始重現 PdM 案例的逐步教學。  
- `reproducibility_checklist.md`：列出為達可重現性所需檢查之項目（資料、環境、版本、隨機種子等）。  
- `diagrams/`：以 Mermaid 語法呈現的架構與流程圖，可直接貼入 GitHub、論文或轉換成 PNG/SVG。
